# README

<h1>This app allows a user to manage his sales, keeping track of the actual state of it.</h1>

Technologies: React and Rails with Webpack

<h2>How to use?</h2>
clone this repo: 
```
 git clone https://github.com/felipedf/edge-store.git 
```

Setup:
```ruby
  bin/setup
```

Run the server:
```ruby
  rails s
```

<h4> Be happy ;)</h4>
